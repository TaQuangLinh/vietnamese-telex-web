# Vietnamese Telex BCI-like Typing Interface

Ứng dụng này mô phỏng giao diện **BCI (Brain–Computer Interface)** với nhiệm vụ phân loại MI thành **3 nhãn**, tương ứng với **3 lệnh điều khiển**: phím **1 – 2 – 3**.  
Mỗi lần nhấn 1/2/3 giúp thu hẹp dần lựa chọn ký tự, tương tự cách BCI điều khiển lựa chọn thông qua phân loại tín hiệu não.

###  Truy cập ứng dụng  
Chỉ cần mở link sau và sử dụng app mô phỏng trực tiếp trên trình duyệt:  
**https://vietnamese-telex-1.onrender.com/**

---

##  Mục đích mô phỏng  

- Mô phỏng cơ chế điều khiển BCI với **3-class MI classification**.  
- Người dùng chỉ dùng **3 phím** để chọn ký tự theo từng tầng.  
- Hệ thống chia ký tự thành 3 nhóm → mỗi lần nhấn phím chọn 1 nhóm → lặp lại cho đến khi còn 1 ký tự.

---

##  Hướng dẫn sử dụng

Ấn vào mục **"? HƯỚNG DẪN"** trong web.
